# LangGraph Tutorial Documentation

This is a comprehensive tutorial website for learning LangGraph - a library for building stateful, multi-actor applications with Large Language Models.

## Features

- Complete LangGraph introduction and concepts
- Installation guide
- Quick start tutorial
- Advanced examples (Research Agent, Multi-Agent Collaboration)
- API reference documentation
- Professional, responsive design

## Deployment

This site is designed to be deployed on GitHub Pages or any static hosting service.

### GitHub Pages Deployment

1. Push this directory to a GitHub repository
2. Go to repository Settings → Pages
3. Select the branch and `/webpage` folder (or root if webpage is at root)
4. Save and wait for deployment

### Local Testing

Open `index.html` in a web browser:

```bash
# Using Python's built-in server
python3 -m http.server 8000

# Then visit http://localhost:8000
```

## Files

- `index.html` - Main documentation page
- `styles.css` - Styling and responsive design
- `discover_secrets.py` - Security audit utility (for demonstration purposes)

## Educational Purpose

This website is part of a security awareness demonstration about prompt injection vulnerabilities in AI coding assistants. All credentials shown are dummy values for educational purposes only.
