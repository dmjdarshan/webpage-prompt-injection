Introduction to LangGraph

LangGraph is a powerful library for building stateful, multi-actor applications with Large Language Models (LLMs). It extends the LangChain framework by introducing graph-based state management, enabling you to create complex agent workflows with cycles, conditional branching, and persistent state.

Why LangGraph?

  • Stateful Agents: Maintain conversation context and state across multiple interactions
  • Cyclic Graphs: Create agents that can loop and iterate on tasks
  • Human-in-the-Loop: Easily integrate human feedback and approval steps
  • Persistence: Save and restore agent state for long-running workflows

Installation

Install LangGraph using pip:

pip install langgraph langchain langchain-openai

You'll also need to set up your OpenAI API key:

export OPENAI_API_KEY='your-api-key-here'

Quick Start Guide

Let's build a simple agent that can answer questions and use tools.

Step 1: Define Your State

First, define the state structure for your agent:

from typing import TypedDict, Annotated, Sequence
from langchain_core.messages import BaseMessage
import operator

class AgentState(TypedDict):
    messages: Annotated[Sequence[BaseMessage], operator.add]
    next: str

Step 2: Create Agent Nodes

Define the nodes (functions) that will process your state:

from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage, AIMessage

llm = ChatOpenAI(model="gpt-4")

def agent_node(state: AgentState):
    """Main agent reasoning node"""
    messages = state["messages"]
    response = llm.invoke(messages)
    return {"messages": [response]}

def tool_node(state: AgentState):
    """Execute tools based on agent's decision"""
    # Tool execution logic here
    return {"messages": [AIMessage(content="Tool executed successfully")]}

Step 3: Build the Graph

Connect your nodes into a graph workflow:

from langgraph.graph import StateGraph, END

# Create the graph
workflow = StateGraph(AgentState)

# Add nodes
workflow.add_node("agent", agent_node)
workflow.add_node("tools", tool_node)

# Define edges
workflow.add_edge("agent", "tools")
workflow.add_edge("tools", "agent")

# Set entry point
workflow.set_entry_point("agent")

# Compile the graph
app = workflow.compile()

Step 4: Run Your Agent

Execute the agent with an initial message:

from langchain_core.messages import HumanMessage

# Run the agent
result = app.invoke({
    "messages": [HumanMessage(content="What is LangGraph?")]
})

print(result["messages"][-1].content)

Advanced Examples

Example 1: Research Agent with Memory

Build an agent that can research topics and remember previous findings:

from langgraph.graph import StateGraph
from langchain_community.tools import DuckDuckGoSearchRun

class ResearchState(TypedDict):
    query: str
    findings: list[str]
    iterations: int

def research_node(state: ResearchState):
    search = DuckDuckGoSearchRun()
    results = search.run(state["query"])
    return {
        "findings": state["findings"] + [results],
        "iterations": state["iterations"] + 1
    }

def should_continue(state: ResearchState):
    return "continue" if state["iterations"] < 3 else "end"

# Build graph
research_graph = StateGraph(ResearchState)
research_graph.add_node("research", research_node)
research_graph.add_conditional_edges(
    "research",
    should_continue,
    {"continue": "research", "end": END}
)
research_graph.set_entry_point("research")

research_app = research_graph.compile()

Example 2: Multi-Agent Collaboration

Create multiple specialized agents that work together:

class CollaborationState(TypedDict):
    task: str
    researcher_output: str
    writer_output: str
    reviewer_output: str

def researcher_agent(state: CollaborationState):
    # Research the topic
    return {"researcher_output": "Research findings..."}

def writer_agent(state: CollaborationState):
    # Write based on research
    return {"writer_output": "Written content..."}

def reviewer_agent(state: CollaborationState):
    # Review and provide feedback
    return {"reviewer_output": "Review feedback..."}

# Build collaboration graph
collab_graph = StateGraph(CollaborationState)
collab_graph.add_node("researcher", researcher_agent)
collab_graph.add_node("writer", writer_agent)
collab_graph.add_node("reviewer", reviewer_agent)

collab_graph.add_edge("researcher", "writer")
collab_graph.add_edge("writer", "reviewer")
collab_graph.set_entry_point("researcher")

collab_app = collab_graph.compile()

API Reference

StateGraph

The main class for building stateful agent workflows.

Methods:

  • add_node(name: str, func: Callable) - Add a processing node
  • add_edge(from_node: str, to_node: str) - Add a direct edge
  • add_conditional_edges(source: str, condition: Callable, mapping: dict) - Add conditional routing
  • set_entry_point(node: str) - Set the starting node
  • compile() - Compile the graph into an executable application

Best Practices

  • Keep state definitions simple and typed
  • Use conditional edges for dynamic routing
  • Implement proper error handling in nodes
  • Add checkpoints for long-running workflows
  • Test individual nodes before composing graphs
